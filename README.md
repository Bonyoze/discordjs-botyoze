## Setup

#### .env
```
BOT_TOKEN=bot's token
BOT_ID=bot's user id
BOT_OWNER_ID=bot owner's user id
```

#### config.json
```json
{
  "intents": [
    list of bot intents
  ]
}
```

Install modules: `npm install`

Start bot: `node index.js`

## Requirements:
[Node.js (v16.6.0 or newer)](https://nodejs.org)
