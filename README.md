# Botyoze
#### A Discord Bot by [Bonyoze](https://github.com/bonyoze) `(Bonyoze#7609)`

WIP

## Setup

#### .env
```
BOT_TOKEN=bot's token
BOT_ID=bot's user id
BOT_OWNER_ID=bot owner's user id
```
#### config.json
```json
{
  "intents": [
    list of bot intents
  ]
}

**Install modules:** `npm install`
**Start the bot:** `node index.js`
```

## Requirements:
